# Facebook-Login-API

The Facebook Login API is basically used to give an easy login step to your portal or website to access it authentically.

fbconfig.php file:- You have to create an app with your facebook account and you need to get the "Facebook App ID" and "Facebook App Secret"

dbconfig.php file:- The file dbconfig is used to make a database connection.

functions.php file:- This step will help you to verify the username and password you entered with username password stored with facebook.

logout.php file: This step will help you to redirect you to a page what ever you want when user click on the logout button of you portal or website. Also this step will destroy you session of connection. 
Further, if you want you can add the cookie expiration methods also.
